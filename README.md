## Descripción del Proyecto
Este sistema integra tres aplicaciones fundamentales de la Inteligencia Artificial desarrolladas en Python con la librería gráfica Tkinter:

1. **P01 - Juego del Gato:** Implementación de algoritmo Minimax para una IA imbatible.
2. **P02 - Juego de Suma 15:** Basado en el isomorfismo de cuadrados mágicos y búsqueda combinatoria.
3. **P03 - Simulador VIM (NIM de Resta):** Estrategia matemática basada en clases de equivalencia (módulo).

Todos los programas permiten:
* Selección de dificultad (Fácil, Medio, Difícil).
* Elección de turno inicial (Jugador, IA o Simulación IA vs IA).
* Registro de marcador en series de 5 partidas.
